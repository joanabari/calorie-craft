# Calorie Craft Project Proposal

## What are you building? Who is it for?

I am building an AI-powered meal planning web application that generates a personalized weekly meal plan based on a user’s calorie deficit goal. This is designed for people trying to lose weight, stay fit, or maintain a healthy diet without spending time planning meals.

## Why?

Many people struggle with maintaining a calorie deficit because meal planning is time consuming and confusing. This project solves that problem by automatically generating daily meal plans tailored to a user’s calorie needs, making it easier to stay consistent and reach fitness goals.

## MVP vs. Stretch Goals

### MVP (Minimum Viable Product):
- User inputs daily calorie target
- App generates a 7 day meal plan
- Each day includes breakfast, lunch, and dinner
- Simple UI with multiple pages (Home, Planner, Results)
- Ability to regenerate meals
- Basic responsiveness for mobile and desktop

### Stretch Goals:
- Macronutrient breakdown (protein, carbs, fats)
- Save favorite meal plans using local storage
- API integration for real recipe data
- Grocery list generator
- Dark/light theme toggle
- AI generated meal suggestions (OpenAI API)

## Technologies

- HTML, CSS, JavaScript
- OpenAI API
